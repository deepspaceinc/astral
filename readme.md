## Contribute

Use `npm` or `bun`:

```bash
bun install
```

To develop locally:

```bash
bun link
bun run dev
```

And in another repo to run Astral against:

```bash
bun link astral
```

Or add it in dependencies in your package.json file:

```bash
bun link astral --save
```
